# Nari Labs - Voz

## 📌 O que faz
Projeto de automação ou agente de IA desenvolvido em Python.

## 🚀 Tecnologias e Stack Usada
- Python

## ⚙️ Como Rodar
1. Clone este repositório e acesse a pasta do projeto.
2. Crie um ambiente virtual (recomendado):
```bash
python -m venv venv
```
3. Instale as dependências necessárias.
4. Execute o arquivo principal:
```bash
python app.py
```

## 🔒 Aviso de Segurança
Este repositório foi higienizado. Todas as chaves de API, senhas, dados de clientes e segredos de negócio (como strings de conexão e URLs internas) foram removidos (`REMOVED_FOR_GITHUB`). Para rodar localmente, adicione suas próprias credenciais em um arquivo `.env` ou nas variáveis de ambiente.
